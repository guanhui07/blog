开发人员能够很好的去应对开发中遇到的问题的关键点在于对问题的判断和索引，大脑的记忆力有限，碎片式记忆需要多练形成潜意识肌肉记忆或者惯性思维，但是惯性思维有可能影响你对问题全面性的判断，所以留下文字，通过对问题的分解和搜索等有效途径去解决问题，这样的方式我觉得更靠谱一些
